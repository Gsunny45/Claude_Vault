# OpenClaw — Agent Framework

## What It Is

Open-source autonomous AI agent framework. 100K+ GitHub stars. Formerly ClawdBot/MoltBot. Node.js 22+ runtime. Connects chat platforms (WhatsApp, Telegram, Slack, Discord, Signal, iMessage, Teams, Matrix, 50+ total) to an autonomous AI agent that executes real-world tasks via LLMs.

## Architecture (5 Components)

1. **Gateway** — Routes messages from chat platforms into unified format
2. **Brain** — Orchestrates LLM calls using ReAct (Reasoning + Acting) loop
3. **Memory** — Persistent Markdown-based context (MEMORY.md + daily logs)
4. **Skills** — 13,700+ plug-in capabilities via ClawHub
5. **Heartbeat** — Task scheduling and proactive inbox monitoring

## ReAct Loop (7 Stages)

Normalize → Route to session → Assemble context → Model inference → ReAct (reason → action → observe → repeat) → Load skills on demand → Persist state

## Built-In Tools

`read`, `write`, `exec` (shell), `web_search`, `web_fetch`, `browser` (Playwright), `video_generate`, `music_generate`

## Memory Model

All state is plain-text Markdown files in agent workspace:
- `MEMORY.md` = long-term facts, preferences, decisions
- `memory/YYYY-MM-DD.md` = daily work logs
- Session isolation for DMs vs group chats
- Memory flush before context compaction

**This maps directly to Obsidian vault structure.** OpenClaw memory files can live inside an Obsidian vault and be indexed/searched/linked like any other note.

## System Requirements

- Node.js 22+ (24 recommended)
- 2+ GB RAM for build
- Docker recommended for production
- Playwright + Chromium for browser automation
- pnpm for dependencies
