## 1. What the project actually is

The "injection keyboard" is a **FlorisBoard fork** (app id `dev.patrickgold.florisboard.vault.debug`, app name "HermeticA-Z") whose purpose is to **inject AI into the typing surface**: API keys, provider ladders, and on-device/cloud model routing, delivered via ADB intents and a JSON orchestration config.

"Injection" has two distinct meanings in this system — keep them separate:

| Sense | What gets injected | Mechanism | Where |
|-------|--------------------|-----------|-------|
| **Key/config injection** | Gemini/provider API keys, provider refs | ADB intent → `CteKeysActivity` (`--ei inject 1 -e keyRef ... -e keyValue ...`) | host PowerShell → device |
| **Prompt/context injection** | vault context, CoT/ToT templates, per-app tone | `skills.json` / `triggers.json` rendered at system-prompt level (KNW-0008 Layer 2) | on-device IME |
