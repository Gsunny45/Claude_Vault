# Claude_on_Claude — Multi-Agent Orchestration Harness

## One-Line Summary

Claude Desktop Pro as the brain, OpenRouter as the model gateway, OpenClaw as the agent runtime, Hostinger VPS as the cloud spine, Obsidian as the memory graph — everything else free.

## The Stack (Paid vs Free)

### PAID (3 services total)
| Service | Cost | Role |
|---------|------|------|
| Claude Desktop Pro | $20/mo | Primary brain, infrastructure builder, MCP host |
| OpenRouter | Pay-per-token (~$5-20/mo) | Multi-model gateway, 300+ models, fallback routing |
| Hostinger VPS KVM 2 | $6.99/mo | Cloud spine: self-hosted n8n, OpenClaw, MCP servers, webhooks |
| **TOTAL** | **~$32-47/mo** | |

### FREE (everything else)
| Service | Free Quota | Role in Stack |
|---------|-----------|---------------|
| OpenClaw | 100% free (open source) | Autonomous agent runtime on Hostinger + Kali |
| Gemini 2.5 Flash | 10 RPM, 250 RPD | Fast general inference, vision, 1M context |

## Default Fallback Chain

```
Request comes in
    ├─ 1. Ollama local (instant, free, always available)
    ├─ 2. Gemini Flash (cheapest free quota)
    ├─ 3. Groq (when speed matters)
    ├─ 4. OpenRouter /free (29+ models, auto-select)
    ├─ 5. Cloudflare Workers AI (edge tasks)
    └─ 6. OpenRouter paid → Claude/GPT-4 (last resort, paid tokens)
```

## OpenClaw Integration

OpenClaw is the autonomous agent runtime. Key facts:
- Open source, 100K+ GitHub stars, Node.js 22+
- 5-component architecture: Gateway → Brain → Memory → Skills → Heartbeat
- ReAct loop: reason → action → observe → repeat
- 13,700+ community skills via ClawHub
- Memory is plain Markdown files (maps directly to Obsidian vault structure)
- Runs on Hostinger KVM 2 ($6.99/mo) or locally on Kali

### OpenClaw Agents (Purpose-Built)
| Agent | Role | Runtime |
|-------|------|---------|
| Pope Bot | TBD — spiritual/philosophical advisor? | OpenClaw on Hostinger |
| Agent0 | Primary autonomous task executor | OpenClaw on Kali |
| OpenClaw core | Base agent with 860+ integrations | Hostinger VPS |
| Hermes | Messenger/router between systems | OpenClaw on Hostinger |

### OpenClaw on Hostinger Setup
1. Hostinger KVM 2 (2 vCPU, 8GB RAM, 100GB NVMe) — $6.99/mo
2. Docker template preinstalled
3. Deploy OpenClaw via Docker Compose
4. Self-hosted n8n alongside in separate container
5. OpenClaw memory files sync to Obsidian vault via REST API
6. n8n webhooks trigger OpenClaw skills

## Gaps Identified

1. **No free tier has production SLAs** — Claude Pro + OpenRouter paid cover this
2. **Local inference is slow** (5-15 tok/s on i5) — use for background/non-urgent only
3. **Storage constraint** (24GB free) — all logging/caching must be aggressive about cleanup
4. **No free fine-tuning anywhere** — use prompt engineering + RAG instead
5. **Gemini free tier reduced 50-80% in Dec 2025** — don't over-depend on it
6. **Copilot CLI free = 50 premium req/mo** — marginal, not a real agent tool
7. **OpenClaw needs Node.js 22+** — must install on Hostinger, not available on Mars's Windows natively (needs WSL or VPS)
