# Free Tier Audit — AI Services (April 2026)

## Tier Ranking (Best Free Value → Worst)

| Rank | Provider | Free Quota | Best Model | Verdict |
|------|----------|-----------|------------|---------|
| 1 | **Ollama (local)** | Unlimited | Llama 3.1 8B Q4 | Always-on, no limits, 5-15 tok/s on i5 |
| 2 | **Groq** | 30 RPM, 1K RPD | Llama 3.3 70B | Best speed, no card needed |
| 3 | **OpenRouter /free** | ~20 RPM, 200 RPD | Qwen3 Coder 480B, 29+ models | Auto-selects from pool |
| 4 | **Gemini Flash** | 10 RPM, 250 RPD | Gemini 2.5 Flash | Vision, 1M context, but quota reduced Dec 2025 |
| 5 | **Cloudflare Workers AI** | 10K neurons/day | Llama 3.2, Whisper, FLUX | Already connected. Edge latency. |
| 6 | **Mistral Experiment** | Rate-limited | Mistral Large, Codestral | Free but data used for training |
| 7 | **HuggingFace** | Few hundred req/hr | All open models | Good for experimentation only |
| 8 | **Copilot CLI** | 50 premium/mo | GPT-4 | Too limited for agent use |
| 9 | **Cohere** | 1K calls/mo | Command R+ | Too restrictive |
| 10 | **Together.ai** | Promo credits only | DeepSeek V3 | No real free tier |

## Combined Free Capacity

~500-1000 requests/day across all providers before hitting limits.

## Recommended Fallback Chain

```
1. Ollama local          → instant, free, offline
2. Gemini 2.5 Flash      → cheapest free cloud quota
3. Groq Llama 3.3 70B    → when speed critical
4. OpenRouter /free       → 29+ models auto-select
5. Cloudflare Workers AI  → edge/vision/speech tasks
6. OpenRouter paid        → Claude/GPT-4 (last resort)
```

## Cost Model

| Scenario | Monthly Cost |
|----------|-------------|
| Free tiers only | $0 (500-1000 req/day max) |
| + Claude Pro | $20/mo |
| + OpenRouter credits | $5-20/mo |
| + Hostinger KVM 2 | $6.99/mo |
| **Full stack** | **$32-47/mo** |

## Key Gaps

- No free tier has SLA or uptime guarantee
- Local inference slow on i5 (5-15 tok/s) — background tasks only
- No free fine-tuning anywhere — use RAG + prompt engineering
- Gemini free tier reduced 50-80% since Dec 2025
- 24GB storage constraint means aggressive cache cleanup needed
- OpenClaw needs Node.js 22+ (install on Hostinger/WSL, not native Windows)
